# Streamers Spotlight

## General info

Application for creating and managing streamers and collecting positive and negative votes for each of them in order to run their popularity ranking.

### Additional info
The application works closely with the API prepared for it, whose repository can be found at this link: [Streamers API](https://github.com/dawidmarek95x/nest_streamers-api)
